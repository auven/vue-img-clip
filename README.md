# vue-img-clip

> vue2.0+canvas图片文字合成，基于[img-clipping](https://github.com/Bless-L/img-clipping)二次开发

[线上地址demo](https://auven.github.io/vue-img-clip/)

## 运行

``` bash
# 安装依赖
npm install

# 开发环境，启动热加载，监听8081端口
npm run dev

# 生产环境，压缩代码
npm run build
```
